내가 클로드로 만든 웹앱이야. 그런데 지금은 각자의 자료를 각자만 볼 수 있어. 하지만 여러 팀원이 쓴 것을 서로가 공유하며 볼 수 있게 하고싶거든. 방법을 알려줄래?

6월 28일

Claude 응답: 좋은 아이디어예요!

좋은 아이디어예요! 현재는 localStorage에 저장하기 때문에 본인 브라우저에서만 데이터가 보이는 구조죠. 팀원끼리 공유하려면 서버나 클라우드에 데이터를 저장해야 해요.

현실적인 방법 3가지를 알려드릴게요.

방법 1. Firebase Realtime Database (추천 ⭐)

무료이고 설정이 가장 간단해요.

흐름:

  1. firebase.google.com에서 프로젝트 생성
  2. Realtime Database 활성화 (테스트 모드로 시작)
  3. 앱에 Firebase SDK 추가
  4. localStorage.setItem(...) 대신 firebase.database().ref('조이름').set(데이터) 로 저장
  5. firebase.database().ref('조이름').on('value', ...) 로 실시간 수신

팀원 모두가 같은 URL로 접속하면 실시간으로 데이터가 공유돼요.

방법 2. 이 앱의 Persistent Storage 활용 (claude.ai 내에서만)

현재 claude.ai 아티팩트 환경이라면 window.storage API를 지원해요. shared: true 옵션을 주면 같은 아티팩트를 보는 모든 사용자가 데이터를 공유할 수 있어요.

javascript
// 저장
await window.storage.set('members', JSON.stringify(data), true); // shared=true

// 불러오기
const result = await window.storage.get('members', true);

단, 같은 아티팩트 링크를 공유한 사람들끼리만 공유돼요.

방법 3. JSON 내보내기/가져오기 (서버 없이 간단하게)

서버 없이 가장 빠른 방법이에요.

  • 내보내기 버튼: 현재 데이터를 .json 파일로 다운로드
  • 가져오기 버튼: 다른 팀원의 .json을 업로드해서 반영

팀원이 자기 데이터를 내보내서 공유 폴더(구글 드라이브 등)에 올리면, 다른 팀원이 불러오는 방식이에요.

어떤 방법이 맞을까요?

FirebasePersistent StorageJSON 내보내기

Firebase 연동을 위해 두 단계가 필요해요.

먼저 Firebase 프로젝트를 세팅해주세요 (5분):

  1. console.firebase.google.com 접속 → 프로젝트 만들기
  2. 왼쪽 메뉴 Realtime Database → 데이터베이스 만들기 → 테스트 모드로 시작
  3. 왼쪽 메뉴 프로젝트 설정(톱니바퀴) → 내 앱 → 웹 앱 추가(</>) → 앱 닉네임 입력 후 등록
  4. 아래처럼 생긴 firebaseConfig 값을 복사해두세요
js
const firebaseConfig = {
  apiKey: "AIza...",
  authDomain: "xxx.firebaseapp.com",
  databaseURL: "https://xxx-default-rtdb.firebaseio.com",  // ← 이게 핵심
  projectId: "xxx",
  ...
};

세팅 완료되면 firebaseConfig 내용을 여기 붙여넣어 주세요 — 바로 코드에 연동해드릴게요!

💡 apiKey는 공개되어도 괜찮아요. Firebase 보안은 Database Rules로 관리돼요.

방법 1. PWA (Progressive Web App) — 추천 ⭐

방법 2. GitHub Pages로 배포 (가장 쉬움)

  1. GitHub 계정 만들기
  2. 저장소(repository) 생성
  3. HTML 파일 업로드
  4. Settings → Pages 활성화
  5. https://유저명.github.io/저장소명 으로 접속 가능

팀원 모두 이 URL로 접속하면 끝이에요.

전체 그림

HTML 파일
    ↓ GitHub Pages에 올리기 (무료)
https://yourname.github.io/repository
    ↓ 팀원 모두 접속
    ↓ Firebase로 데이터 실시간 공유
    ↓ 폰 홈화면에 설치 (PWA)

Firebase 연동 → GitHub Pages 배포 → PWA 설치까지 순서대로 도와드릴 수 있어요. Firebase 설정 먼저 완료되면 알려주세요!

  1. 제가 드린 파일 링크 위에 오른쪽 클릭
  2. "다른 이름으로 저장" 클릭해서 컴퓨터에 저장
